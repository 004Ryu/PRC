    Abstract

    Para este TPC foi-nos proposta a criação de um DataSet em formato JSON de nadadores ou swimmers através dos DataSets da DBpedia. Para tal utilizei a seguinte query de SPARQL: 

                select distinct ?swimmer ?name ?gender ?birthDate ?birthPlace ?deathDate ?abstract where {
               ?swimmer a dbo:Swimmer.
               ?swimmer foaf:name ?name.
               ?swimmer foaf:gender ?gender.
               ?swimmer dbo:birthDate ?birthDate.
               
               OPTIONAL { ?swimmer dbo:birthPlace ?birthPlace.}
               OPTIONAL { ?swimmer dbo:deathDate ?deathDate.}
               OPTIONAL { 
                  ?swimmer dbo:abstract ?abstract.
                  FILTER (lang(?abstract) = 'en')
               }
            }
